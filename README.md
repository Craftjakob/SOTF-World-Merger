# SOTF-World-Merger
This is a Sons of the Forest World Merger. It attempts to merge two SaveData.zip files so that you can use your structures from other worlds in a single world.

---
### WARNING
*This "World Merger" is **NOT** recommended for merging worlds with the intention of playing afterwards on them.
While it is capable of merging worlds almost correct and allows for proper interaction with many of the structures. But issues may arise with certain structures, such as rope bridges or I think, the very first structure built. For instance, during snowfall or rainfall, weird squares may be displayed where neither snow nor rain is actually visible.*
